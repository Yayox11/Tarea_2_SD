Instrucciones de ejecución:

La(s) torre de control fue programado en Python, el(los) avión(es) fue(ron) programado(s) en Ruby y la pantalla de información
fue programada en Node.js.

Para ejecutar la torre de control, dentro de la carpeta Server, ejecute python server.py, para el avión, dentro de la carpeta
Server, ejecute ruby client.rb y para la pantalla de información, ejecute npm install para instalar las dependencias y
node client.js para ejecutar la pantalla. 

Comentarios sobre tarea:

-Para el avión, no se puede consultar en la puerta acaso cumple con las restricciones de combustible y pasajeros.
-La torre logra consultar los aviones, pero por alguna razón los muestra como undefined, problema que no alcanzamos a resolver.
-Al comienzo, se debe especificar la IP y el puerto a utilizar, tanto en la torre como en el avión.
